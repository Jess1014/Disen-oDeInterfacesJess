Elementos de html

El término "elemento" son las partes que componen el documento HTML y esta conformado por:

  • Una etiqueta de apertura
  • Cero o más atributos
  • Texto encerrado por la etiqueta
  • Una etiqueta de cierre

El texto encerrado es opcional porque algunas etiquetas no encierran el texto.

La estructura de un elemento HTML comienza con una etiqueta de apertura ( < p>) que contiene uno o ningun atributo, contiene texto(Prueba de html) y finaliza con la etiqueta de cierre ( < /p>). A continuación se muestra una tabla de algunos elementos por categoria.

-Elemento raíz- -
< !doctype html> Define que el documento es bajo el estandar de HTML y representa la raíz de un documento HTML o XHTML.
-Metadatos del documento- -
< head> Representa una colección de metadatos acerca del documento.
< title> Define al documento con un título que se muestra en la barra de título.
< base> Define la URL base para las URLs relativas en la página.
< link> Es usada para enlazar JavaScript y CSS esternos con el documento HTML actual.
< meta> Define los metadatos que no pueden ser definidos usando otro elemento HTML.
< style> Es la etiqueta de estilo usada para escribir CSS en línea.
-Secciones- -
< body> Representa el contenido principal del documento.
< section> Define una sección en un documento.
< h1> - < h6> Son los elementos de cabecera que implementan 6 niveles donde < h1> es el mayor y < h6> el menor.
< main> Define el contenido principla o importante en el documento.
-Agrupación de contenido- -
< blockquote> Pone el texto en sangría.
< figure> Representa una figura dentro del documento.
< p> Define una parte como párrafo.
< ol> Define una lista ordenada.
< ul> Define una lista enumerada.
< dt> Define un término.
< dd> Es la definición o descripción.
-Semántica a nivel texto- -
< br> Es un salto de página.
< u> Subraya el texto.
< b> Pone texto en negrita.
< strong> Define que es importante y lo pone en negrita .
< mark> Marca el texto.
< i> Pone el texto en cursiva.
< cite> Define el título y lo pone en cursiva.
< dfn> Representa el elemento definición y lo pone en cursiva.
< em> Enfatiza y coloca en cursiva el texto.
< q> Pone el texto entre comillas.
< small> Define el texto con letra más pequeña.
-Ediciones- -
< del> Define el texto que a sido borrado.
< ins> Define el texto que a sido insertado subrayandolo.
-Contenido incrustado- -
< iframe> Representa un contenido anidado de navegación.
< img> Inserta una imagen.
< map> En conjunto con < area> define un mapa en el area.
< area> En conjunto con < map> define un mapa en el area.
-Datos tabulares- -
< table> Representa datos conmás dimensión.
< th> Representa una celda encabezado en una tabla.
< tr> Representa una fila de celdas en la tabla.
< td> Representa una celda de datos en la tabla.
